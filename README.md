### 👨🏻‍💻 Como executar o projeto

[Node.js](https://nodejs.org/) v16 ou superior para executar.

Execute os comandos abaixo para instalar as depêndencias do projeto e execução de testes:

```sh
cd teste-front-vr
npm i
npx cypress run
```
